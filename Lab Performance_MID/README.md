# Product Management Project
 
I completed this project by building a simple Product Management page where a user enters quantity per day and sees the total price automatically.
 
## What I did
 
1. Created the page layout with input fields for quantity and total price.
2. Styled the page using CSS so it looks clean and easy to use.
3. Added JavaScript logic to calculate total price with this formula:
 
    **Total Price = Unit Price x Quantity per Day x Days**
 
    Here, Unit Price = 1000 and Days = 30.
4. Added validation so quantity cannot be negative.
5. Added a coupon alert feature: if total price is more than 1000, a message shows that the user is eligible for a gift coupon.
